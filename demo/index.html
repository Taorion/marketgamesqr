Guion comercial

  1. Mostrar que la campana puede iniciar en juego, TikTok, evento o formulario.
  2. Generar un QR y explicar que el token no contiene datos faciles de falsificar.
  3. Abrir el validador, iniciar sesion y pegar el QR.
  4. Mostrar cedula y telefono para comparar con el cliente.
  5. Redimir y volver a validar el mismo QR para mostrar bloqueo de doble uso.
